# Deep Hedging — 신경망 기반 옵션 헤지

## 한 줄 요약
거래비용이 존재하는 현실 시장에서, **신경망이 직접 헤지 전략을 학습**하게 한다. Black-Scholes 델타 헤지를 베이스라인으로 두고, 거래비용·이산 시간 하에서 딥러닝 정책이 이를 능가하는지 검증한다. (Buehler et al., 2019, "Deep Hedging" 재현·확장)

## 왜 Sales & Trading 인가
- 옵션을 팔면(세일즈) 그 리스크를 헤지(트레이딩)해야 한다. 이게 데스크의 본업.
- 교과서 BS 델타 헤지는 **연속 시간·무거래비용** 가정. 현실은 이산 시간 + 비용 + 마찰. 그래서 실무 헤지는 늘 근사.
- Deep Hedging은 이 현실 제약을 **목적함수에 직접 넣고** 최적 정책을 학습 → 모던 퀀트의 대표 사례. 면접·포트폴리오 화제성 최고.

## 핵심 아이디어
- 헤지를 **순차적 의사결정 문제**로 본다: 매 시점 보유량을 얼마로 바꿀까?
- 신경망 정책 π(상태) → 헤지 포지션. 상태 = 현재 가격, 보유량, 잔존만기 등.
- 손실 = 만기 시점 포트폴리오 손익의 **위험척도**(예: CVaR / mean-variance) + 누적 거래비용.
- 경로(price path)들을 시뮬레이션 → 신경망이 비용 포함 위험을 최소화하도록 end-to-end 학습.

## 목표 & 산출물
1. 가격 경로 시뮬레이터 (GBM 우선, 다음 단계로 변동성 모델 확장).
2. 옵션 페이오프 + 거래비용 모델.
3. **베이스라인**: BS 델타 헤지 (이산 시간, 비용 포함).
4. **Deep Hedging**: 신경망 정책 학습 (CVaR 등 위험척도 목적함수).
5. 비교 리포트: 거래비용 0 / 中 / 高 시나리오에서 손익 분포·CVaR·거래비용 비교. "비용이 커질수록 Deep Hedging이 이긴다"를 그래프로.

## 시장·상품 설정 (1차 버전)
- 기초자산: GBM 시뮬레이션 (μ, σ 고정). σ 는 **앞 프로젝트(Volatility Forecasting)** 추정값 사용 가능.
- 상품: 유럽형 콜옵션 1개 (vanilla). 매도 포지션을 헤지.
- 시간: 만기까지 N 스텝 이산. 매 스텝 리밸런싱.
- 거래비용: 거래대금 비례 비용 (proportional cost). 강도 파라미터로 시나리오 분기.

## 모델
| 구성 | 내용 |
|------|------|
| 베이스라인 | BS 델타 헤지 (이산, 비용 포함) |
| 정책망 | MLP 또는 시점별 가중치 공유 RNN. 입력=상태, 출력=헤지 비율 |
| 목적함수 | CVaR_α(최종 P&L) 또는 mean-variance. 거래비용 포함 |
| 학습 | 시뮬레이션 경로 배치 → SGD (Adam). 경로는 매 에폭 새로 생성 |

## 평가 방법
- 동일 테스트 경로 세트에서 두 전략 비교.
- 지표: 최종 P&L 분포(히스토그램), 평균/표준편차, **CVaR**, 거래비용 총액.
- 거래비용 강도 sweep: 0, 0.1%, 0.5%, 1% → Deep Hedging 우위가 어디서 생기나.
- (확장) 변동성 비정상·점프 추가 시 강건성.

## 마일스톤
1. **M1 — 환경**: GBM 시뮬레이터, 콜옵션 페이오프, 거래비용 모델. BS 가격/델타 해석해 검증.
2. **M2 — BS 베이스라인**: 이산 델타 헤지 + 비용. P&L 분포 산출. 비용 0일 때 분포가 좁아지는지 확인(정합성 체크).
3. **M3 — Deep Hedging 골격**: 정책 MLP, CVaR 손실, 학습 루프. 비용 0에서 BS 델타에 수렴하는지 확인(필수 sanity check).
4. **M4 — 비용 도입**: 거래비용 sweep. Deep Hedging이 BS를 이기는 지점 찾기.
5. **M5 — 확장 & 리포트**: 변동성 비정상/점프 등 추가, 결과 시각화, 결론 문서화.

## 기술 스택
- Python 3.11+, numpy, pandas
- `pytorch` (정책망·학습)
- `scipy` (BS 해석해 검증), `matplotlib` (P&L 분포 시각화)

## 사전 지식 / 참고
- Buehler, Gonon, Teichmann, Wood (2019), *Deep Hedging*.
- Black-Scholes 델타 헤지, CVaR(조건부 VaR) 개념.
- 앞선 **Volatility Forecasting** 프로젝트의 변동성 추정 결과를 시뮬레이션 σ 입력으로 연결.
